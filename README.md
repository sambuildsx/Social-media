# Social Media app

A modern social media web app combining real-time chat (like Discord) and content sharing (like Instagram).

## Features
- User authentication
- Real-time chat
- Posts & feeds
- Follow / unfollow system
- Responsive UI

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Real-time: Socket.io

## 📦 Setup

```bash
git clone <your-repo-link>
cd your-project
npm install
npm start
